# E-Commerce-Business-Intelligence-Data-Analysis-
This project focuses on analyzing and optimizing the performance of an e-commerce company using Python-based Exploratory Data Analysis (EDA) and an interactive Power BI Business Intelligence Dashboard.

This project focuses on analyzing and optimizing the performance of an e-commerce company using Python-based Exploratory Data Analysis (EDA) and an interactive Power BI Business Intelligence Dashboard.

The primary objective is to transform raw transactional data into actionable insights that improve financial performance, customer satisfaction, logistics efficiency, and product management.

Using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn, the dataset was cleaned and analyzed. Missing values in the Amount column were handled using median imputation, duplicates were removed, and categorical variables were standardized to ensure accurate analysis.

The Power BI dashboard was developed to provide real-time, interactive insights into:

📊 Financial Performance – Revenue trends, average order value, and best-selling products

👥 Customer Insights – B2B vs B2C behavior, top-performing locations, and cancellation trends

🚚 Logistics & Fulfillment – Order status distribution, shipping service comparison, and fulfillment performance

📦 Product Management – High-demand categories, SKU performance, and quantity trends

🔁 Customer Satisfaction – Return and cancellation rate analysis

The dashboard enables business stakeholders to filter data by date, state, category, and fulfillment type, supporting data-driven decision-making.

The objective is to optimize:
- Financial performance
- Customer insights
- Logistics & fulfillment
- Product management
- Customer satisfaction & returns

The project includes:
✔ Python-based Exploratory Data Analysis (EDA)  
✔ Data Cleaning & Preprocessing  
✔ Interactive Power BI Business Intelligence Dashboard

The e-commerce company is facing:

## 📈 Key Insights

- Identified high cancellation categories.
- Merchant fulfilled orders had higher delay rates.
- Certain states contribute majority of revenue.
- Seasonal revenue spikes observed.
- B2B customers show higher average order value.

## 🚀 Business Recommendations

- Improve logistics for merchant-fulfilled orders.
- Optimize inventory for top-performing SKUs.
